# 🎯 Number Guessing Game (Java)

A simple console-based **Number Guessing Game** built in Java. The program generates a random number between **1 and 100**, and the player has to guess it within a limited number of attempts.

---

## 📌 Features

- 🎲 Generates a random number between **1 and 100**
- ⌨️ Takes user input using `Scanner`
- 📈 Provides hints:
  - Too High
  - Too Low
- ✅ Displays a success message when the correct number is guessed
- 🔢 Limits the number of attempts
- 🎮 Beginner-friendly Java project

---

## 🛠️ Technologies Used

- Java
- Scanner Class
- Random Class
- Loops (`while`)
- Conditional Statements (`if-else`)

## 🚀 How to Run

1. Clone the repository

```bash
git clone https://github.com/your-username/Number-Guessing-Game.git
```

2. Navigate to the project folder

```bash
cd Number-Guessing-Game
```

3. Compile the program

```bash
javac NumberGame.java
```

4. Run the program

```bash
java NumberGame
```
